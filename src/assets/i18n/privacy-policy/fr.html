2. Remarques générales

Cette déclaration de protection des données explique la nature, l'étendue et le but de la collecte et de l'utilisation des données personnelles par l'application "Listen". Je prends très au sérieux la protection de vos données personnelles et je respecte strictement les règles des lois sur la protection des données.

3. Collecte et traitement des données

3.1. Aucune collecte de données personnelles

Mon application ne recueille aucune donnée personnelle des utilisateurs. Toutes les données saisies sont stockées exclusivement sur l'appareil de l'utilisateur. Aucun stockage ou traitement de ces données sur mes serveurs n'a lieu.

3.2. Stockage des données dans Google Cloud

Si l'utilisateur a activé une sauvegarde de ses données, celles-ci seront stockées dans Google Cloud. Cela ne se produit que si l'utilisateur l'a expressément activé dans les paramètres de l'application. Les politiques de confidentialité de Google sont accessibles à l'adresse https://policies.google.com/privacy.
La responsabilité de la sécurité des données dans Google Cloud incombe à Google.

3.3. Transmission des données aux appareils Garmin

Les données que vous saisissez sont également transmises via la Garmin ConnectIQ API aux montres connectées Garmin via Bluetooth dans le cadre de l'utilisation de l'application. Cette transmission des données n'a lieu qu'entre votre smartphone et la montre Garmin et ne sera pas stockée ou traitée sur mes serveurs.

4. Utilisation de AdMob

4.1. Généralités

Mon application utilise AdMob, un service de Google Inc., pour intégrer des publicités. AdMob peut collecter et traiter des données personnelles telles que l'adresse IP et l'identifiant publicitaire. Ces données sont utilisées pour afficher de la publicité personnalisée.

4.2. Type de traitement des données par AdMob

AdMob utilise des cookies, des balises web et des technologies similaires pour diffuser des annonces basées sur vos visites précédentes dans cette application et d'autres applications ou sites web. Les informations collectées peuvent inclure :

  • Informations sur l'appareil (par exemple, fabricant, modèle, système d'exploitation)
  • Identifiants uniques (par exemple, identifiant publicitaire)
  • Adresse IP
  • Informations de localisation

4.3. Objectif du traitement des données par AdMob

Les données collectées par AdMob sont utilisées pour diffuser des annonces publicitaires personnalisées en fonction de vos intérêts. Cela se fait sur la base de votre consentement conformément à l'article 6, paragraphe 1, point a) du RGPD.

4.4. Suppression des données AdMob

En tant qu'utilisateur, vous avez le droit de demander la suppression des données collectées par AdMob. Puisque AdMob est exploité par Google, le traitement des données est effectué directement par Google. Pour supprimer vos données, suivez ces étapes :

  • Visitez les paramètres de confidentialité et de sécurité de Google.
  • Connectez-vous avec le compte Google associé aux données collectées.
  • Gérez vos paramètres publicitaires, supprimez votre ID d'annonces ou demandez la suppression des données selon les instructions de Google.

Veuillez noter que, en tant que développeur, je n'ai pas accès direct aux données collectées par AdMob et ne peux donc pas initier la suppression de ces données. Pour obtenir une assistance supplémentaire, vous pouvez contacter le support AdMob.

4.5. Politique de confidentialité d'AdMob

Pour plus d'informations, consultez la politique de confidentialité d'AdMob : Politique de confidentialité AdMob.

5. Déclaration de consentement

Lors du premier démarrage de l'application, votre consentement pour l'utilisation d'AdMob et le traitement des données associées sera sollicité. Vous avez le droit de révoquer ce consentement à tout moment. Dans les paramètres de l'application, vous pouvez désactiver l'affichage des annonces publicitaires personnalisées.

6. Vos droits

Vous avez le droit d'accéder à vos données personnelles, de les corriger, de les supprimer et de limiter leur traitement, ainsi que le droit de transférer ces données et de vous opposer à leur traitement. Pour exercer ces droits, vous pouvez me contacter à tout moment.

6.1. Droit à l'information

Vous avez le droit de demander une confirmation que des données personnelles vous concernant sont traitées et, le cas échéant, d'obtenir des informations sur ces données ainsi qu'une copie de celles-ci.

6.2. Droit de rectification

Vous avez le droit de demander la rectification de données personnelles incorrectes vous concernant. Compte tenu des finalités du traitement, vous avez également le droit de demander que les données personnelles incomplètes soient complétées.

6.3. Droit à l'effacement

Vous avez le droit de demander que les données personnelles vous concernant soient effacées immédiatement, à condition qu'une des raisons suivantes s'applique:

  • Les données personnelles ne sont plus nécessaires au regard des finalités pour lesquelles elles ont été collectées ou traitées d'une autre manière.
  • Vous retirez le consentement sur la base duquel le traitement était effectué conformément à l'article 6, paragraphe 1, point a), du RGPD, et il n'existe pas d'autre fondement juridique pour le traitement.
  • Vous vous opposez au traitement et il n'existe pas de motifs légitimes impérieux pour le traitement.
  • Les données personnelles ont fait l'objet d'un traitement illicite.

6.4. Droit à la limitation du traitement

Vous avez le droit de demander que le traitement de vos données personnelles soit limité si l'une des conditions suivantes est remplie:

  • Vous contestez l'exactitude des données personnelles, pendant une durée permettant de vérifier l'exactitude de ces dernières.
  • Le traitement est illicite et vous vous opposez à leur effacement en demandant à la place la limitation de leur utilisation.
  • Les données personnelles ne sont plus nécessaires aux fins du traitement, mais elles sont encore nécessaires pour la constatation, l'exercice ou la défense de droits en justice.
  • Vous avez fait valoir un droit d'opposition au traitement pendant la vérification portant sur l'existence de motifs légitimes impérieux.

6.5. Droit à la portabilité des données

Vous avez le droit de recevoir les données personnelles vous concernant que vous nous avez fournies, dans un format structuré, couramment utilisé et lisible par machine, et vous avez le droit de transmettre ces données à un autre responsable du traitement sans que nous y fassions obstacle.

6.6. Droit d'opposition

Vous avez le droit de vous opposer à tout moment, pour des raisons tenant à votre situation particulière, au traitement des données personnelles vous concernant effectué sur la base de l'article 6, paragraphe 1, points e) ou f), du RGPD. Je ne traiterai alors plus vos données personnelles, à moins que je puisse démontrer qu'il existe des motifs légitimes et impérieux pour ce traitement qui prévalent sur vos intérêts, droits et libertés, ou pour la constatation, l'exercice ou la défense de droits en justice.
