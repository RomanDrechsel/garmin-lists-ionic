2. General Information

3. Data Collection and Processing

3.1. No Collection of Personal Data

My app does not collect personal data from users. All data entered is stored exclusively on the user's device. There is no storage or processing of this data on my servers.

3.2. Data Storage in Google Cloud

If the user enables a backup of their data, the data will be stored in Google Cloud. This only occurs if the user explicitly activates this feature in the app settings. Google's privacy policies can be found at https://policies.google.com/privacy.
The responsibility for data security in Google Cloud lies with Google.

3.3. Data Transfer to Garmin Devices

The data you enter is also transferred to connected Garmin watches via Bluetooth using the Garmin ConnectIQ API as part of the app usage. This data transfer occurs solely between your smartphone and the Garmin watch and is not stored or processed on my servers.
